Overview

Yuer DSL is a next-generation, expression-driven decision language designed to control AI cognitive systems with deterministic, auditable, and structured behavior.

It is the primary language layer of EDCA-OS (Expression-Driven Cognitive Architecture), enabling:

deterministic task execution

semantic-layer control

stable reasoning chains

auditable cognitive behavior

conflict-free multi-step operations

Unlike prompt-based instructions, Yuer DSL defines explicit execution structures, control metadata, and behavioral contracts that guide the model safely and predictably.
